About

I build iOS apps end to end — design, code, App Store listing, support, all of it. Everything is written in Swift, mostly with programmatic UIKit, because I like having full control over how a screen behaves and feels.

Most of my apps share a small set of building blocks I've refined across all 16: card-based layouts with real depth instead of flat drop shadows, consistent onboarding flows, and a design language that holds up whether the app is a puzzle game or a budgeting tool.
